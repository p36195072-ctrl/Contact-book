Contact Book Manager

A beginner-friendly Python project that demonstrates how to create a contact management system using JSON files and dictionaries.

📌 Description

This project allows users to:

Add contacts
Search contacts
View all contacts
Save contacts permanently
Load contacts automatically
Exit safely

All contact information is stored in contactbook.json, so data remains available even after closing the program.
